Gemini HUD is a modern, lightweight, always-on-top utility for Windows that provides instant access to [Google Gemini](https://gemini.google.com). It features a frameless design, global hotkeys, and persistent sessions.

## âœ¨ Features

- **Instant Access**: Toggle the HUD anytime with `Alt + G`.
- **Frameless Design**: Clean, modern UI with no native window borders.
- **Custom Controls**: Integrated drag region, minimize, and hidden-close behavior.
- **Persistent Sessions**: Stay logged in to Gemini across restarts.
- **Responsive**: Resizable window that remembers your workflow.

## ðŸš€ Quick Start

1. **Download**: Get the latest installer from the `dist/` folder (or build it yourself).
2. **Install**: Run `gemini-hud-1.0.0-setup.exe`.
3. **Launch**: Open the app. It runs in the background.
4. **Use**: Press `Alt + G` to show/hide the window.
